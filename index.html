📲 Install App on Your Device
Install Alora for offline access, faster loading, and a home screen icon
📱 iPhone / iPad:
1. Open this page in Safari
2. Tap the Share button ⬆️ (bottom center bar)
3. Scroll down → tap "Add to Home Screen"
4. Tap Add — the app icon appears on your home screen
🤖 Android:
1. Open this page in Chrome
2. Tap the ⋮ menu (top right)
3. Tap "Add to Home screen" or "Install app"
💻 Laptop / Desktop:
1. Open in Chrome or Edge
2. Click the install icon ⊕ in the address bar (right side)
3. Or go to Menu → Install Alora Property Tracker
4. The app opens in its own window — no browser bars
Works on iPhone, iPad, Android, Mac, Windows, and Chromebook
📡 How Online & Offline Sync Works
✅ Online: All changes sync instantly to every device via Firebase. Everyone sees updates in real-time.
📴 Offline: You can still view all unit data, update statuses, add notes, and browse maps. Changes save to your device locally.
🔄 Reconnect: When you get internet back, everything syncs automatically. No data is lost.
🤖 AI Chat: Requires internet (calls the API). Everything else works fully offline.

📖 Quick Reference Guide
Status Icons
✅ Occupied — Tenant in unit, rent-ready
⬜ Vacant — Empty, awaiting action
⚡ In Progress — Brushworx actively working
🔨 Needs Demo — Demolition required before make-ready
🔧 Make Ready Done — Unit is prepped and ready for rent / management review
🏗️ Put Back — Reconstruction after demo
⛔ Down Unit — Offline, major work needed
🏆 BW Complete — Brushworx finished, ready to lease
Dashboard Cards
Tap any card on the Home screen to see a filtered list of those units. From the list you can search, select multiple units, batch-update statuses, or export PDFs.
Map View
🔍 Use the filter bar to highlight buildings by status, tier, or bedroom count. Tap a building to see its units in the sidebar. Tap a unit to edit status, tenant, tier, notes, or flag it as urgent.
Tiers (A–E)
🟢 A — Minor touch-up   🔵 B — Light repairs   🟡 C — Moderate work   🟠 D — Heavy rehab   🔴 E — Full gut / mold
Data & Sync
All changes sync automatically between devices. Use Export CSV for spreadsheet backups, or Download Backup for full JSON snapshots. The 🔔 activity bell shows recent team changes.
